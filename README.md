# ⚡ EliteCart

A modern **E-commerce Shopping Cart Website** built using **HTML, CSS, and JavaScript**.

EliteCart showcases premium tech products with a clean UI and a fully functional cart system.

## 🚀 Features
- Responsive modern UI
- Product listing with categories
- Add to Cart functionality
- Quantity increase / decrease
- Remove items from cart
- Dynamic cart total
- Cart sidebar interface
- LocalStorage support (cart saved after refresh)

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
